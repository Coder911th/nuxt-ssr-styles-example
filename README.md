# Demo Nuxt.js SSR <style/> error
New Nuxt.js versions don't render <style/> tags in the template area (page/index.vue).
## 2.9.2
``` bash
cd 2.9.2
npm i
npm run build
npm start
```
Open http://localhost:3000

The rectangle is red.

## 2.11.0
``` bash
cd 2.11.0
npm i
npm run build
npm start
```
Open http://localhost:3000

The rectangle is white.
